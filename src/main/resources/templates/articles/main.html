게시판 링크 리스트

url method 출력내용 html page
/articles/list GET 전체 게시글 읽기 성공 list_all.html
/articles/{id} GET 7번 게시글 읽어오기 성공 list_one.html
/articles/create?name=tom&weight=8 0&height=180 GET [[${name}]]의 키는 [[${height}]], 몸무게는 [[${weight}]] 입니다. new.html
/articles/update POST 업데이트 성공 화면 출력 update_ok.html
/articles/{id}/update GET [[ ${id} ]] 번 게시글 수정 폼 로딩 완료 update.html
/articles/{id}/delete GET [[${deleteId}]]번 글 삭제 완료 delete_ok.html
/articles/{id}/articleComment POST [[${articleId}]]번 게시글의 모든 댓글보기 성공 comment_view.html
/articles/{id}/articleComments/{articlecomment-id}/delete POST [[${articleId}]]번 게시글의 [[${commentId}]]번째 답글 삭제 완료 delete_ok.html
